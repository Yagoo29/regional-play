# Regional Play

Plataforma web desenvolvida em HTML, CSS e JavaScript para disponibilização de conteúdo esportivo em uma interface moderna, responsiva e otimizada para dispositivos móveis.

## Recursos

* Interface moderna e responsiva
* Compatível com celulares, tablets e computadores
* Área de transmissão integrada
* Integração com Kiwify para pagamentos
* Design otimizado para eventos esportivos
* Hospedagem gratuita via GitHub Pages
* Fácil personalização

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages
* Kiwify

## Como Executar Localmente

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/RegionalPlay.git
```

2. Entre na pasta do projeto:

```bash
cd RegionalPlay
```

3. Abra o arquivo `index.html` no navegador.

## Publicação no GitHub Pages

1. Faça upload dos arquivos para o GitHub.
2. Acesse **Settings → Pages**.
3. Em **Source**, selecione:

   * Branch: `main`
   * Folder: `/root`
4. Salve as alterações.

Seu site ficará disponível em:

```text
https://SEU-USUARIO.github.io/RegionalPlay/
```

## Integração com Kiwify

Substitua os links de exemplo pelos links reais dos seus produtos na Kiwify:

```html
href="SEU_LINK_KIWIFY"
```

Após a compra, o cliente poderá ser redirecionado para a área de acesso da plataforma.
